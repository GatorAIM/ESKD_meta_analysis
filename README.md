# Machine Learning–Based Outcome Prediction in Dialysis Patients  
**Systematic Review and Meta-Analysis**

This repository contains the data and code used for the meta-analysis in:

> **Systematic Review and Meta-Analysis of Machine Learning–Based Health Outcome Prediction in Dialysis Patients with Kidney Failure**  
> Qi Xu, Ho Yin Chan, Hongyu Chen, Lauren E. Adkins, Ashutosh M. Shukla, Mei Liu  


The analyses focus on synthesizing discrimination performance (AUROC/C-statistic) of machine learning models predicting mortality and non-mortality outcomes in patients receiving dialysis.
